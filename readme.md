# API Request Checker Tool

## Overview

This is a simple web-based tool built using HTML, CSS, and JavaScript that allows users to test API requests. It provides an interface to send both GET and POST requests to specified APIs and displays the response.

## Features

- Send GET requests to APIs and display the response.
- Send POST requests to APIs with custom data and display the response.
- Simple and intuitive user interface.

## How to Use

1. Clone this repository to your local machine:


2. Open the `index.html` file in your preferred web browser.

3. Enter the API endpoint URL in the provided input field.

4. Choose the request method (GET or POST).

5. If making a POST request, enter the JSON data in the textarea provided.

6. Click the "Send Request" button to see the response.

## Example APIs to Test

Here are some example free APIs you can use to test the functionality of this tool:

### GET Requests

1. **Random User Generator API**
   - URL: https://randomuser.me/api/
   - Description: Generates random user data.

2. **JSONPlaceholder**
   - URL: https://jsonplaceholder.typicode.com/posts/1
   - Description: Returns a sample post.

### POST Requests

1. **ReqRes.in**
   - URL: https://reqres.in/api/users
   - Description: Create a new user by sending a POST request with user data.

2. **JSONPlaceholder**
   - URL: https://jsonplaceholder.typicode.com/posts
   - Description: Create a new post by sending a POST request with post data.

Feel free to use these APIs to test the functionality of the tool.
